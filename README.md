# Snake

3D self-playing snake game

### Develop

```
npm install
npm start
```

### Build

```
npm run build
```

## Deploy

Add a `.env` file to the project root with the following info:

```
DEPLOY_HOST=
DEPLOY_PATH=
DEPLOY_PORT=
DEPLOY_USER=
```

Then run:

```
npm run deploy
```
